UNITYCHAT

Browser extension · v3.12

Sjednocený live chat z Twitch, YouTube a Kick v jednom side panelu. 7TV / BTTV / FFZ emoty, odesílání zpráv, raids, pin, user cards. Chrome · Edge · Opera · Brave.

Stáhnout UnityChat.zip v… · updated …

Co to umí

3 platformy najednou

Twitch IRC, YouTube live chat API, Kick Pusher WebSocket. Vše v jednom feedu řazené chronologicky, s platform badges.

7TV / BTTV / FFZ

Všechny populární 3rd-party emoty + native Twitch a Kick. Tab autocomplete pro rychlé psaní.

Odesílání zpráv

Odpovídej do chatu na kterékoli ze tří platforem z jednoho input fieldu. Reply context se propaguje automaticky.

Raids, pins, first-msg

Raid notifikace, pin management přes Twitch GQL, first-time chatter highlight, auto-scroll pause.

UnityChat marker

Uživatelé s extensionem mají oranžový gradient badge — poznáš ostatní UnityChat uživatele na první pohled.

Jak nainstalovat

UnityChat se distribuuje self-hosted — není v Chrome Web Store. Instalace přes Developer mode trvá asi 2 minuty.

1

Stáhni ZIP

Klikni na tlačítko Stáhnout UnityChat.zip nahoře. Soubor se uloží do Downloads.

2

Rozbal ho

Rozbal ZIP a složku si přesuň někam na stálo (třeba Dokumenty\UnityChat\).

3

Otevři extensions stránku

V prohlížeči napiš do URL baru:

  • chrome://extensions pro Chrome
  • edge://extensions pro Edge
  • opera://extensions pro Opera
  • brave://extensions pro Brave
4

Zapni Developer mode + Load unpacked

Vpravo nahoře přepni Režim pro vývojáře, vlevo se objeví tlačítko Načíst rozbalené (Load unpacked).

5

Vyber složku

Klikni Načíst rozbalené, vyber tu složku kam jsi ZIP rozbalil — tu co obsahuje přímo manifest.json.

6

Otevři UnityChat

Ikona UnityChat se objeví v toolbaru (nebo v 🧩 puzzle menu — odtud ji pinni). Chrome/Edge/Brave: klik otevře side panel. Opera: popup window, nebo si UnityChat přidej přes Customize sidebar pro native sidebar integraci.

Updaty

Když vydám novou verzi, stáhni znovu ZIP, rozbal přes starou složku a v chrome://extensions klikni na 🔄 reload u UnityChat karty. Hotovo, nemusíš znovu Load unpacked.

FAQ

Proč to není v Chrome Web Store?

Store má specifická pravidla ohledně ukládání uživatelských dat, která budou relevantní pro plánovaný cloud chat archiver. Pro teď distribuuju self-hosted abych neřešil policy gating. Pokud bude setup v plné konformitě, přesunu na store.

Je to bezpečné?

Zdrojový kód je veřejný na github.com/jouki/UnityChat — můžeš si ho přečíst celý. Extension se připojuje jen na Twitch, YouTube, Kick a jejich emote CDN (7TV, BTTV, FFZ, IVR badges). Žádná telemetrie, žádné analytiky, žádný tracking.

Funguje to na Firefoxu?

Ne. Never say never, ale zatím není v plánu.

Proč musím povolit tolik permissions?

sidePanel pro side panel UI · cookies pro čtení Twitch auth-token (nutné pro odesílání zpráv přes GQL) · scripting pro content scripty v chat stránkách · tabs pro detekci aktivní platformy · storage pro nastavení + cache · downloads pro export debug logu.

Jak nahlásit bug?

Otevři issue na GitHub Issues. Přidej prosím výstup z 💾 debug log button v hlavičce extensionu — pomůže rychleji najít příčinu.
