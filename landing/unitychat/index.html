UNITYCHAT

Sjednocený live chat z Twitch, YouTube a Kick v jednom side panelu. 7TV / BTTV / FFZ emoty, odesílání zpráv, raids, pin, user cards. Chrome · Edge · Opera · Brave.

Jak to vypadá

Živý mockup sestavený z reálného sidepanel.css extensionu — to co vidíš je přesně to co dostaneš po instalaci.

Co to umí

Funkce v aktuální verzi.

3 platformy najednou

Twitch IRC (WebSocket), YouTube live chat API, Kick Pusher WebSocket. Všechny v jednom feedu, řazené chronologicky, s platform badges.

7TV / BTTV / FFZ

Všechny populární 3rd-party emoty + native Twitch a Kick. Tab autocomplete pro rychlé psaní.

Odesílání zpráv

Odpovídej do chatu na kterékoli ze tří platforem z jednoho input fieldu. Reply context se automaticky propaguje.

Raids, pins, first-msg

Raid notifikace, pin management přes Twitch GQL, first-time chatter highlight, auto-scroll pause.

Opera native sidebar

V Opeře se dá přidat do levého sidebaru (Customize → UnityChat). V Chrome native side panel. Runtime feature detection.

User cards

Klik na username otevře Twitch user card nebo draggable floating fallback s founded/followed datem a rolí.

UnityChat marker

Uživatelé s extension mají v chatu oranžový gradient badge — poznáš ostatní UnityChat uživatele na první pohled.

Debug log export

💾 tlačítko v headeru stáhne debug log do Downloads složky. Hodí se pro bug reporty.

Jak nainstalovat

UnityChat není v Chrome Web Store, distribuuje se self-hosted. Instalace přes "Load unpacked" v Developer mode trvá asi 2 minuty.

  1. 1

    Stáhni ZIP

    Klikni na tlačítko Stáhnout UnityChat.zip nahoře. Soubor se uloží do Downloads.

  2. 2

    Rozbal ho

    Přesuň ZIP do stálé složky (třeba Dokumenty\UnityChat\) a rozbal. Nemaž ani nepřesunuj tu složku později — Chrome ztratí extension a přestane fungovat.

  3. 3

    Otevři extensions

    V prohlížeči napiš do URL baru:

    • chrome://extensions pro Chrome
    • edge://extensions pro Edge
    • opera://extensions pro Opera
    • brave://extensions pro Brave
  4. 4

    Zapni Developer mode

    Vpravo nahoře je přepínač Developer mode. Zapni ho.

  5. 5

    Load unpacked

    Objeví se tlačítko Load unpacked (nebo Načíst rozbalené). Klikni a vyber složku kam jsi ZIP rozbalil — tu co obsahuje manifest.json.

  6. 6

    Otevři UnityChat

    Ikona UnityChat se objeví v toolbaru (nebo v 🧩 puzzle menu, odkud si ho pinni). Chrome/Edge/Brave: klik otevře side panel. Opera: popup window, nebo si přidej přes Customize sidebar pro native Opera sidebar.

Pozor na Developer mode warning: Chrome ti při startu občas ukáže varování "Disable developer mode extensions?". Klikni Cancel / Zrušit, jinak UnityChat vypne. Je to generický Google nag, ne problém s tímto extensionem.

Jak získat updaty

Když vydám novou verzi, stáhni znovu ZIP z tohoto webu a překopíruj obsah do stejné složky (nebo ji smaž a rozbal čerstvě). Pak v chrome://extensions klikni na 🔄 reload u UnityChat extensionu. Hotovo.

FAQ

Proč to není v Chrome Web Store?

Store má specifická pravidla ohledně ukládání uživatelských dat která by mohla narážet na plánovaný cloud chat archiver (48h catch-up feature). Pro teď distribuuju self-hosted, abych neřešil policy gating. Pokud to jednou bude v plné konformitě, přesunu na store.

Je to bezpečné?

Zdrojový kód je veřejný na github.com/jouki/UnityChat — můžeš si ho přečíst celý. Extension se připojuje jen na Twitch, YouTube, Kick a jejich emote CDN (7TV, BTTV, FFZ, IVR badges). Žádná telemetrie, žádné analytiky, žádný tracking.

Funguje to na Firefoxu?

Zatím ne. Firefox má mírně odlišnou manifest syntax (browser_specific_settings, některé permissions). Na roadmapě je Firefox podpora jako separate adaptace.

Ukládá to moje zprávy někam na server?

Lokálně ano — malá cache v chrome.storage.local pro rychlé zobrazení po reloadu (max 200 zpráv, auto-rotace). Na žádný server nic neposílá. Plánovaný cloud sync bude vždy opt-in v settings.

Proč musím povolit tolik permissions?

sidePanel pro side panel UI · cookies pro čtení Twitch auth-token (nutné pro odesílání zpráv, Helix API to vyžaduje) · scripting pro content scripty v chat stránkách · tabs pro detekci aktivní streamovací platformy · storage pro uložení nastavení a cache · downloads pro export debug logu.

Jak nahlásit bug?

Otevři issue na GitHub Issues. Přidej prosím výstup z 💾 debug log button v hlavičce extensionu — pomůže mi to najít příčinu rychleji.
